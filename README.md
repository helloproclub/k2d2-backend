# k2d2 Backend

![Travis](https://travis-ci.org/helloproclub/k2d2-backend.svg?branch=master)

Requirement:

- Node v8.x
- Redis
- MongoDB

Tested On:

- Ubuntu 16.04
- Node 7.6.0
- Redis 4.0.1
- MongoDB 3.2.12

This software requires Node > 7.6.0 to support Async/Await.

# Usage

1. run `npm install`
2. run `npm run dev`
3. Authenticate bitrix by accessing `hostname:port/auth`
