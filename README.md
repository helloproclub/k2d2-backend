# k2d2 Backend

![Travis](https://travis-ci.org/helloproclub/k2d2-backend.svg?branch=master)

Requirement:

- Node >= 7.6.0
- Redis
- MongoDB

Tested On:

- Ubuntu 16.04
- Node 7.6.0
- Redis 4.0.1
- MongoDB 3.2.12

# Setup
1. Create `.env` file, sample can found in `.env.example`
2. run `npm install`
3. run `npm run dev`
4. Authenticate bitrix24 by accessing `hostname:port/auth`
5. Profit!!
