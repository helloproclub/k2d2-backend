# k2d2 Backend
![Travis](https://travis-ci.org/helloproclub/k2d2-backend.svg?branch=master)

Requirement:
- Node v8.x
- Redis
- MongoDB

Tested On:
- Ubuntu 16.04
- Node v8.2.1
- Redis 4.0.1
- MongoDB 3.2.12

# Usage
1. run `npm install`
2. run `npm run dev`
3. Authenticate bitrix by accessing `hostname:port/auth`
